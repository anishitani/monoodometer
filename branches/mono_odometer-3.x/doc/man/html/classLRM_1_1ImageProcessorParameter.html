Detailed Description

List of parameter regarding ROS configuration for the launch file.

  • FEATURE_TYPE: Type of feature to be detected. (Default: SIFT)
  • MAX_NUM_FEATURE_PTS: Maximum number of features to be detected. (Default: 100)
  • MATCH_RADIUS: Radius used as a filter for the feature match. (Default: 10.0)
